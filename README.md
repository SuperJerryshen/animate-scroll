# Animate-Scroll

> 使用 Animate-Scroll，可以利用动画过渡至指定高度

## 安装

```bash
npm install --save @jerryshen520/animate-scroll
```

## 使用

```javascript
import { scrollToY } from '@jerryshen520/animate-scroll';

scrollToY(0);
scrollToY(1000);
```

## API

### scrollToY

- 参数(`YCoord`: `number`, `config`: `object`)
